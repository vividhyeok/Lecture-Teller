# LectureTeller 단축키

StreamDeck류 매크로 버튼에 그대로 옮기기 쉽도록 현재 기준 단축키를 정리한 문서입니다.
입력창(`input`, `textarea`, `select`)에 포커스가 없을 때 동작합니다.

## 재생 제어
- `Space` : 재생 / 일시정지
- `ArrowLeft` : 10초 뒤로
- `ArrowRight` : 10초 앞으로
- `Shift` + `ArrowLeft` : 5초 뒤로
- `Shift` + `ArrowRight` : 5초 앞으로
- `[` : 이전 문장 또는 현재 문장 처음
- `]` : 다음 문장

## 볼륨 / 배속
- `ArrowUp` : 볼륨 10% 증가
- `ArrowDown` : 볼륨 10% 감소
- `Shift` + `ArrowUp` : 배속 0.25 증가
- `Shift` + `ArrowDown` : 배속 0.25 감소
- `0` : 배속 1.0x 초기화
- `M` : 음소거 토글

## 파일 이동
- `Shift` + `N` : 다음 파일
- `Shift` + `P` : 이전 파일

## 탭 / 화면
- `V` : 자료 -> 대본 -> 학습 탭 순환
- `C` : 대본 탭 바로 이동
- `1` : 싱크 뷰
- `2` : 요약 뷰
- `3` : 집중 뷰
- `4` : 블라인드 뷰

## 학습 액션
- `B` : 핵심 북마크 추가
- `H` : 헷갈림 북마크 추가
- `F` : 암기 포인트 추가
- `L` : 현재 문장 반복 토글
- `R` : 헷갈림 큐 토글

## 보기 크기
- `+` / `=` : 글자 확대
- `-` / `_` : 글자 축소
